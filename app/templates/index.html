Welcome to YEET

Yet anothEr mEtagenomics Tool

YEET is a web based tool for analysis of metagenomics data. Several attributes can influence the quality of reads in metagenomics datasets. YEET allows interactive filtering and visualization of metagenomics data to detect and remove low quality reads. Visualization includes:

  • Sequence length
  • Distribution of nucleotides
  • Percentage of paired reads
  • Identity score between forward and reverse complement sequences


YEET uses FASTQ files as input, which will undergo preprocessing to determine attributes for each read. By using visualization and filtering specific attributes, low quality reads can be cleared from the data.
