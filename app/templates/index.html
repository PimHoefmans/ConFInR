Welcome to ConFInR

ConFInR is a web-based tool for functional annotation of metagenomics data. It includes pre-processing methods which allow interactive filtering and visualization of the data to detect and remove low quality reads. Some of the attributes that can be filtered are:

  • Sequence length
  • Distribution of nucleotides
  • Percentage of paired reads
  • Identity score between forward and reverse complement sequences
DIAMOND, a high performance sequence aligner, is used to align metagenomics sequences to proteins which in turn can be used for functional annotation.

More information can be found on GitHub.
